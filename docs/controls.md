# 操作方法(PC / スマホ差分表)

JIB-FREAK MOBILE の操作一覧。第5回生徒会で決定。

| 操作 | PC(キーボード) | スマホ(タッチ) |
| ---- | -------------- | -------------- |
| 開始 | スペース / Enter | 画面をタップ |
| ステージ選択 | ↑↓ | STAGE 1 / STAGE 2 の行をタップ |
| 移動 | 矢印キー | 画面のどこでも指を置いてドラッグ(相対移動) |
| 射撃 | スペース(押しっぱなしで連射) | 触れている間は自動射撃 |
| 連射の最速 | スペースの物理連打(新規押下は即発射) | タップ連打 |
| ポーズ | P | 画面右上の ⏸ ボタン |
| 音のON/OFF | M | 画面右上の ♪ ボタン(既定はOFF) |
| ショット変更 | Z (隠しコマンド) | なし |
| スピード変更 | S (隠しコマンド) | なし |
| ボム | B (隠しコマンド) | なし |

- 画面内のUI文言は英語表記(大掃除#1・レトロゲーの雰囲気重視)。
- 1面をクリアするとスコアと装備を持ち越して2面へ続く(大掃除#1・
  元の設計の実現)。ハイスコア狙いは1面から通しが基本線。
- 隠しコマンドは意図的にPC限定(第5回の会長答弁)。
- ホールド連射は約0.15秒間隔。新規の押下は間隔を待たずに撃てるため、
  最速の連射は物理的な連打で出る(会長発議・アーケードの伝統)。
