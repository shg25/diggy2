# 生徒会議事録

JIB-FREAK MOBILE の新機能は、生徒会で審議して実装する。
その様子を書記が議事録として記録したのがこのディレクトリである。

- **授業**([lessons/](../lessons/)) では過去のコードから学んだ。
  各レッスン末尾の「放課後」は、あとから直した居残り作業の記録
- **生徒会**(ここ)では未来の機能を審議し、実装する

## 議事録の形式

1機能 = 1回の会合 = 1つの議事録。各議事録には以下を記す。

- **議題**: 何を作るか
- **審議**: なぜ作るか、どう作るかの検討(却下した案も残す)
- **決議**: 決めたこと
- **実施報告**: 実装の説明と対応コミット
- **書記所感**: 会合を記録していて思ったこと

## 会員名簿

- **会長**: ゲームの持ち主。議題の採否と方針を決める
- **顧問**: 実装を補佐する教員(授業のときの先生と同一人物)
- **書記**: この議事録を書いている
